# RPG Game

This is a small Role-Playing game that was made using Gamemaker Studio 2 IDE and the GML Language.

Features:
Combat system
Inventory system

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

To load the project, install the Gamemaker Studio 2 IDE and load the project folder into a local machine. To test the program, run it within the IDE.

### Installation

```
1. Install the GameMaker Studio 2 IDE (https://www.yoyogames.com/get)
2. Open the project from the unzipped folder
3. Run from IDE
```
